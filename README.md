# TransMenu Website

這是 iOS App **TransMenu** 的官方 landing page 與政策頁面。網站以靜態 HTML/CSS/JS 建構，可直接部署於 GitHub Pages 或任何支援靜態檔案的主機。

## 開發指南

- `index.html`：主要的 landing page，包含產品特色、故事介紹與 CTA 區塊。
- `policy.html`：隱私權政策與使用條款。
- `assets/styles.css`：全站共用的風格設定。
- `assets/script.js`：互動邏輯（菜單雷達、CTA 表單動態等）。

## 如何啟動

只要使用任一靜態伺服器（如 `python -m http.server`）或直接開啟 `index.html` 即可預覽，部署至 GitHub Pages 時無須額外設定。
